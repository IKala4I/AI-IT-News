Розуміє текст, зображення, відео й документи. Alibaba заявляє, що модель "друга після Fable 5" — але це поки суто їхні власні оцінки, незалежних бенчмарків ніхто не публікував.

Прев'ю (Qwen3.8-Max-Preview) вже доступне через Token Plan, Qoder і QoderWork за пільговими цінами, і працює через OpenAI- і Anthropic-сумісні API — можна підключити хоч у Claude Code, хоч у Cursor. Повні відкриті ваги обіцяють "скоро", дати нема.

Реліз вийшов рівно за тиждень після Kimi K3 — Alibaba явно поспішали застовпити місце в гонці великих опенсорсних моделей, поки бенчмарки ще нікого не спростували.

## 🛠️ Інструменти

**Claude Code [оновився до 2.1.215](https://raw.githubusercontent.com/anthropics/claude-code/main/CHANGELOG.md) — і заразом з'ясувалось, що всередині тула вже давно Bun, переписаний на Rust**

Сама 2.1.215 — крихітна: Claude більше не запускає скіли `/verify` і `/code-review` сам по собі за власною ініціативою, тільки коли покликати їх явно.

Цікавіше знайшов Simon Willison: [пороївшись у бінарнику](https://simonwillison.net/2026/Jul/19/claude-code-in-bun-in-rust/) (`strings` + пошук рядків `Bun v1.4.0` і 563 файлів `.rs`), він підтвердив, що з версії 2.1.181 Claude Code ганяє на Bun, переписаному на Rust — ще до публічного релізу цієї версії Bun назовні. Це й дало непомітний +10% до старту на Linux, про який майже ніхто не написав.

Гарна ілюстрація того, як великі інфраструктурні зміни тихо просочуються в тул, яким користуєшся щодня, без жодного анонсу в реліз-нотах.

**[transcribe.cpp](https://workshop.cjpais.com/projects/transcribe-cpp) — новий локальний ASR-рушій на ggml, 60+ моделей розпізнавання мови в одному бінарнику**

Підтримує 16 родин моделей (Whisper, Parakeet, Canary, GigaAM, Moonshine, Qwen3-ASR, SenseVoice та інші) — кожну звірили по word error rate з оригінальною реалізацією. Працює на Mac/Windows/Linux, прискорення через Vulkan, Metal, CUDA чи навіть на слабких SBC в реальному часі. MIT-ліцензія, є біндинги для Python, JS/TS, Rust, Swift.

```bash
cmake -B build -DTRANSCRIBE_VULKAN=ON
cmake --build build
build/bin/transcribe-cli -m models/parakeet-tdt-0.6b-v2/parakeet-tdt-0.6b-v2-F32.gguf samples/jfk.wav
```

На HN уже 650+ балів — якщо треба локальний speech-to-text без прив'язки до одного сімейства моделей і без хмари, виглядає як серйозна альтернатива whisper.cpp.
