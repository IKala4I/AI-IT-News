### Anthropic’s Fable 5 & Mythos 5 Still Suspended Globally — Still Developing

On June 12, the US Commerce Department issued an export-control directive ordering Anthropic to suspend all access to its two newest frontier models, Claude Fable 5 and Mythos 5 (released just three days earlier on June 9), for any foreign national — including Anthropic’s own employees. Because Anthropic cannot reliably identify foreign nationals in real time, the result is a hard global shutoff. The trigger was reportedly a claim by another company that it had jailbroken Mythos; Anthropic disputes this, calling it a likely misunderstanding and describing any observed jailbreak as narrow and non-universal. The company is complying while working to restore access.

**Why it matters:** This is unprecedented: a government ordering an AI lab to shut off its flagship products globally, less than 72 hours after launch. It sets a live precedent for export-control treatment of AI models and raises questions about how frontier AI will be regulated going forward.

## 💻 IT / General Tech News

### Fireship: SQLite Being Rewritten in Rust (Turso/Limbo)

Fireship published a video (June 19) covering Turso’s ongoing rewrite of SQLite in Rust — a project codenamed “Limbo.” The Rust-based version adds concurrent writes (a long-standing SQLite limitation), an async-first API, native vector search, Change Data Capture, and improved full-text search. Turso reached a milestone in early 2026: the first Rust-based version supporting concurrent writes shipped. The Rust rewrite is currently in beta, with Turso’s production service still running libSQL (a C-based SQLite fork).

**Why it matters:** SQLite is embedded in billions of devices and applications — it’s arguably the most widely deployed piece of software on Earth. A Rust rewrite would make it memory-safe and add capabilities critical for edge computing and AI workloads (vector search, async, concurrent access).

Sources: [Fireship video](https://www.youtube.com/c/Fireship) · [Turso blog](https://turso.tech/blog/we-are-a-year-into-rewriting-sqlite) · [Hacker News discussion](https://news.ycombinator.com/item?id=46810950)

## 🛠️ New Tools & Releases

### Cloudflare: Temporary Accounts Let AI Agents Deploy Without Signing Up

Cloudflare launched “Temporary Cloudflare Accounts for AI Agents” (June 19). Any agent can now run `wrangler deploy --temporary` and have a live Worker deployed in seconds — no account creation, no OAuth flow, no MFA prompt. The temporary instance stays live for 60 minutes, during which a human can “claim” it into a permanent account. Cloudflare also launched a companion feature with Stripe: AI agents can autonomously provision full Cloudflare accounts, purchase domain names, start paid subscriptions, and deploy to production — all without human intervention, using a new co-designed agent-to-payment protocol.

**Why it matters:** The biggest bottleneck for autonomous AI agents acting in the real world is authentication — systems built for humans that require browsers, CAPTCHA, and MFA. Cloudflare and Stripe are building the first pieces of an “agentic internet” where software can act like a first-class customer. Expect other platforms to follow.

Sources: [Cloudflare blog](https://blog.cloudflare.com/temporary-accounts/) · [Cloudflare + Stripe agents](https://blog.cloudflare.com/agents-stripe-projects/) · [Changelog](https://developers.cloudflare.com/changelog/post/2026-06-19-temporary-accounts-for-agents/)

### ArgusRed: Post-Trained Pentest Model That Exploits, Doesn’t Just Flag

Cosine launched ArgusRed (trending on Hacker News today), an automated penetration testing service powered by a custom security-tuned model post-trained to “think like an adversary.” The key differentiator: it doesn’t produce a list of potential vulnerabilities — it attempts to actually reproduce exploits in an isolated sandbox and only reports confirmed breaches, along with the full request/response proof and an auto-generated pull request with the fix. Pricing: $100 per repository, fully refunded if no exploits are confirmed. Code and sandboxes run in UK/EU by default.

**Why it matters:** Most security scanners drown teams in false positives. Confirming exploits before reporting them is a genuine step change in signal-to-noise for security teams. The “refund if we find nothing” pricing model is aggressive market positioning.

Sources: [ArgusRed](https://argusred.com) · [Hacker News](https://news.ycombinator.com/)

### Simon Willison: Datasette-Apps Plugin — Custom HTML Apps Inside Datasette

Simon Willison released `datasette-apps`, a Datasette plugin that lets developers host custom sandboxed HTML applications directly within a Datasette instance using iframes with content-security-policy controls. He also released `datasette-acl` (alpha), expanding ACL/permission features toward general resource-sharing for multi-user instances. He also noted on his blog that MCP’s (Model Context Protocol) real advantage over traditional skills/CLIs is **isolating auth flows outside the agent’s context window** — a useful mental model for anyone building agentic systems.

**Why it matters:** Datasette is becoming a serious platform for data publishing and lightweight apps. The auth-isolation framing for MCP is also worth bookmarking: it clarifies *why* MCP is architecturally useful beyond just being a “plugin system.”
