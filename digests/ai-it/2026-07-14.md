## 🤖 AI Models & Research

### Satya Nadella Warns Enterprises Are "Paying for Intelligence Twice"

Microsoft CEO Satya Nadella published an essay arguing that companies using proprietary AI vendors pay twice: once in token fees, and again by leaking their own proprietary know-how into a vendor's model through prompts, corrections, and everyday usage — what he calls the "reverse information paradox," since AI vendors freely scrape the public web to train but restrict customers from doing the same back to them. He recommends enterprises build their own "proprietary learning environments" and orchestration layers rather than depending entirely on closed frontier APIs.

**Why it matters:** Coming from the CEO of a company invested in both OpenAI and Anthropic, this is a notable break from unqualified AI cheerleading — effectively an argument for hedging toward open-weight or on-prem alternatives, likely to resonate with enterprise IT buyers weighing vendor lock-in.

### Telegram's t.me Domain Suspended at the Registry Level, Breaking Links Worldwide

Registry-level "server hold" status suspended Telegram's core t.me domain, removing it from global DNS and breaking all t.me invite/channel/message links worldwide (the app itself kept working). Reports point to a compliance action tied to US OFAC sanctions requirements at the .me registry; no official cause has been confirmed. **Why it matters:** Shows how a single domain-registry action can sever a major platform's public-facing links globally overnight, and also disrupted Telegram's TON crypto ecosystem, which relies on t.me links as an on-ramp.

### 148 Malicious npm Packages Turned Visiting Browsers Into a DDoS Botnet

A second wave of 148 npm packages disguised as "student proxy" tools for bypassing school web filters hid a remote script loader (fetching unvetted code from a mutable GitHub branch via jsDelivr, no integrity checks) and a WebSocket-flood module that opened 1,024 connections per visiting browser to attack targets, including a small Illinois nursing school's CDN. The campaign originally launched in March as adware, added malicious modules in May, and relaunched July 8 as this larger second wave; most packages have since been pulled, though a couple of versions were still serving live malicious code as of July 14.

**Why it matters:** Another case of the open-source supply chain being weaponized against unwitting end users (students, in this case) rather than developers directly — and the infrastructure remains re-armable with a single commit.

Source: [The Hacker News](https://thehackernews.com/2026/07/148-npm-packages-disguised-as-student.html)

### Ant Group Open-Sources SingGuard-NSFA, a Real-Time Guardrail Model for AI Agents

Ant Group's AI Safety Lab open-sourced SingGuard-NSFA, a purpose-built guardrail model that sits in front of autonomous AI agents and screens their actions before execution — checking for prompt injection, sensitive-data theft, malicious code execution, resource abuse, and permission misuse. It covers 185 threat scenarios across 7 categories in 133 languages, ships in four sizes (0.8B–9B parameters), and returns a risk judgment in about 50ms, with the compact 0.8B model reportedly matching some 8B competitors.

**Why it matters:** As coding and browsing agents move from suggesting actions to autonomously executing them, this is one of the first open-source, purpose-trained guardrail models sized for real-time production use — relevant to anyone building agent frameworks rather than repurposing a general LLM as an ad-hoc judge.
